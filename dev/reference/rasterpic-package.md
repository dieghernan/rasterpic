# rasterpic: Convert Digital Images to Spatially Referenced 'SpatRaster' Objects

Convert digital images to spatially referenced 'SpatRaster' objects, as
defined by the 'terra' package, using coordinates from supported spatial
input classes. Supported inputs include numeric coordinate vectors and
objects from the 'sf', 'terra' and 'stars' packages. The main function
is an S3 generic, allowing other packages to extend support to
additional spatial classes.
